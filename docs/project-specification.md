# Spertu Astro Demo Project Specification

Version: 0.1  
Last Updated: 2026-03-08

## Goal
Build a reusable Astro demo site that demonstrates how a real small business / agency website can be structured for Spertu CMS.

## Included
- Homepage with structured sections
- Case studies collection and detail pages
- Team data and site settings data sources
- Spertu configuration contract
- Local Git-ready project structure

## Constraints
- Keep dependencies minimal
- Preserve clear separation between editable content and non-editable UI code
- Prioritize readability and maintainability over abstraction

## Success Criteria
- Site builds successfully with Astro
- Content lives in explicit predictable paths
- `spertu.config.ts` reflects real editable areas
- Project is ready to be pushed to a new GitHub repository
