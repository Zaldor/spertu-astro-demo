# Spertu Astro Demo Design Bible

Version: 0.1  
Last Updated: 2026-03-08

## Product Intent
Create a polished Astro demo that feels credible for a modern service brand while proving how Spertu CMS can safely manage structured website content.

## UX/UI Direction
- Modern agency aesthetic with premium restraint
- Strong typography and spacious layout hierarchy
- Calm dark interface with soft gradients and elevated glass-like panels
- Clear section rhythm: hero, services, case studies, team, CTA
- Responsive behavior that preserves clarity on smaller screens

## Design System Rules
- 8px-based spacing rhythm
- Rounded medium-to-large surfaces
- Accent gradients reserved for key CTAs and highlight panels
- Muted secondary text to preserve reading hierarchy
- High-contrast focus-friendly controls

## Content Modeling Rules
- Pages use Markdown frontmatter for structured section content
- Case studies use Astro content collections
- Team and site settings use JSON for straightforward structured editing
- Components remain developer-owned and non-editable

## Compatibility Goal
Every editable surface should map cleanly into a future Spertu editing UI without requiring schema inference.
